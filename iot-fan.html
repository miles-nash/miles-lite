IoT Smart Fan

Built a smart office fan using ESP32 microcontroller, 8x8 thermal camera, ambient temperature sensor, rotary encoder, and neopixel ring. Controlled fan with PWM using BJT transistor, implemented basic feedback control system, and published data wirelessly to HVAC control system using MQTT protocol.

Integrated thermal imaging for targeted cooling, ambient temperature monitoring for environmental awareness, and rotary encoder for manual control. System provides both automated and manual control modes with real-time data transmission to building management systems.
