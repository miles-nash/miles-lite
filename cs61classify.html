RISC-V Neural Network Classifier

Project Specification

Implemented a neural network classifier for handwritten digit recognition on RISC-V architecture. The project involved designing and optimizing neural network inference algorithms to run efficiently on the RISC-V processor.

Key features include optimized matrix multiplication routines, efficient memory management for neural network weights, and real-time classification of MNIST dataset digits. The system demonstrates the capabilities of RISC-V architecture for machine learning applications.
