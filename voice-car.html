Voice Controlled Car

Project Writeup | Demo Video

Class project implementing a voice-controlled autonomous vehicle system using the SIXT33N car platform. Built complete voice recognition and motor control systems from the ground up, integrating microphone boards, signal processing filters, and classification algorithms for autonomous navigation.

Developed analog and digital interfaces including DAC/ADC circuits, implemented motor controller circuits with speed sensors, and created voice classification using PCA and cluster classification algorithms. Applied discrete state-space control via eigenvalue placement for closed-loop motor control, enabling the car to drive straight and turn in response to voice commands. Comprehensive integration of signal processing, controls theory, and machine learning in embedded systems design.
